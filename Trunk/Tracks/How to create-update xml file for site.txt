1. Файлы генерятся автоматически генератором
2. Для успешной работы генератора надо в .gpx файле у тега gpx вырезать описание namespace
3. Если исохдый .gpx меняется, то перегенерить надо в случаях:
	- изменилось количество треков
	- изменилось количество точек
	- изменились сами треки
4. И не надо в случаях:
	- изменились имена треков и точек, иконки, линки
